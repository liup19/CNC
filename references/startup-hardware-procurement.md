# 创业公司硬件采购策略

> 最后更新：2026-05-16

---

## 核心区别

```
个人的痛点：便宜、方便
创业公司的痛点：品控、一致性、可复制、可量产
```

---

## 三个阶段

```
阶段 1 — 原型验证（1-5 台）
  目标：证明技术方案可行
  策略：能买到什么就用什么
  渠道：淘宝、AliExpress、零售

阶段 2 — 小批量试产（10-100 台）
  目标：验证产品能不能批量做
  策略：找小工厂合作
  渠道：1688、工厂直联、深圳周边

阶段 3 — 量产（100+ 台）
  目标：控制成本、保证品质
  策略：深度绑定供应商
  渠道：代工厂、OEM/ODM
```

---

## 阶段 1：原型验证

### 核心原则

```
花钱买时间，不要自己造轮子

做法是买成品拆解 + 改装：

  买 3 台 3018 整机（淘宝 ¥500 × 3 = ¥1500）
  拆开研究结构、理解每个零件的作用
  然后在此基础上改装：
    - 换控制板（运行自己的固件）
    - 加传感器（力传感器、温度传感器）
    - 改装主轴（加转速反馈）

  不从零开始设计硬件
  → 先用现成硬件验证软件/AI 方案是否可行

┌──────────────────────────────────────────────┐
│ AI CAM 创业的核心价值在软件，不在硬件        │
│                                              │
│ 硬件：买现成的，改装测试                      │
│ 软件：AI 参数推荐 + 刀路策略 + 用户交互       │
│                                              │
│ 硬件差异化是后期的事                          │
│ 初期不应该在硬件上花太多时间和钱              │
└──────────────────────────────────────────────┘
```

### 原型阶段采购清单

```
硬件测试：
  3018 整机 ×2         ¥1000    淘宝
  Shapeoko 整机 ×1     ¥5000    淘宝/官方
  各种刀具套装         ¥200     淘宝
  各种材料（铝/木/亚克力）¥300    淘宝

电子开发：
  ESP32 开发板 ×5      ¥100     淘宝
  传感器套件           ¥200     淘宝
  逻辑分析仪/示波器    ¥300-500 淘宝（调试用）

软件/云服务：
  LLM API 费用         ¥200-500/月
  服务器               ¥100-300/月

原型阶段总投入：约 ¥8000-12000
```

## 阶段 2：小批量试产

### 核心原则

```
从"买东西"变成"找合作伙伴"

问题：淘宝零件每批质量不一样
  这次买的丝杠精度 ±0.02mm
  下次买同一家同款 ±0.08mm
  → 做出来的机器性能不一致 → 用户投诉
```

### 找稳定供应商

```
方法：在 1688 上找工厂（不是贸易商）

判断标准：
  - 店名带"XX五金厂""XX机械厂"（工厂直营）
  - 店名带"XX贸易""XX商贸"（中间商，避开）
  - 有"实力商家""源头工厂"标签
  - 营业执照看经营范围（有"生产"二字）

谈判要点：
  "我前期每次拿 20-50 件，后续量会增长，
   能不能给我一个稳定的价格和质量标准？"

工厂角度：
  20-50 件太少，大部分工厂不接
  但有些小厂产能不满 → 愿意接小单
  关键：承诺后续增长 → 建立长期关系
```

### 签质量协议

```
关键件（丝杠、导轨、主轴）必须有：
  - 尺寸公差标准（如丝杠精度 C7 级）
  - 抽检比例（每批抽 10% 检测）
  - 不合格品的退换条款
  - 材质证明（铝合金牌号 6063-T5 等）
```

### BOM 固定

```
把所有零件的规格、型号、供应商固定下来：

┌───┬──────────────┬──────────────┬──────────┬────────┐
│ # │ 零件         │ 规格         │ 供应商    │ 单价   │
├───┼──────────────┼──────────────┼──────────┼────────┤
│ 1 │ 滚珠丝杠     │ SFU1605-300  │ XX精密   │ ¥85    │
│ 2 │ 线性导轨     │ MGN12H-300   │ XX工贸   │ ¥65    │
│ 3 │ 步进电机     │ 23HS22-2804S │ XX电机厂 │ ¥45    │
│ 4 │ 控制板       │ 自研 PCB     │ 嘉立创   │ ¥35    │
└───┴──────────────┴──────────────┴──────────┴────────┘

BOM 固定后：
  - 每台机器用料一样 → 质量一致
  - 可以精确计算成本 → 定价有依据
  - 可以批量采购 → 拿到更低价
```

## 阶段 3：量产

### 方案 A — 找代工厂（OEM）

```
把整个生产外包给一家工厂：
  你提供：设计图纸 + BOM + 质量标准
  工厂负责：采购零件 + 组装 + 测试 + 包装

去哪里找代工厂：
  深圳周边（东莞、惠州）→ 消费电子类代工厂多
  浙江（义乌、温州）    → 五金机械类工厂多
  阿里 1688 "找工厂" 功能
  线下展会（深圳国际机械展、CME 上海机床展）

代工成本结构（以一台 ¥3000 零售价的 CNC 为例）：

  BOM 物料成本:   ¥800-1000   (30-35%)
  组装人工:       ¥100-150    (3-5%)
  工厂利润:       ¥100-200    (5-7%)
  包装物流:       ¥50-80      (2-3%)
  ─────────────────────────
  出厂成本:       ¥1050-1430  (35-48%)

  你的毛利:       ¥1570-1950  (52-65%)
  扣掉研发/营销/售后 → 净利 15-25%
```

### 方案 B — 部分外包 + 自己组装核心模块

```
标准件（铝型材、丝杠、导轨）：代工厂采购 + 加工
电子产品（控制板）：自己设计 → 嘉立创打板 + 贴片
核心组装（整机组装 + 烧录固件 + 测试）：自己团队做

好处：控制核心技术（软件 + 控制板）
坏处：需要自己的组装场地和工人

适合 AI CAM 创业公司：
  核心价值是软件 → 硬件可以大部分外包
  只保留：
    - 控制板设计（自己的固件需要匹配硬件）
    - 整机测试（确保软件运行正常）
    - 固件烧录（自己的软件装进去）
```

### 方案 C — 完全自己生产（不推荐初期）

```
需要厂房、设备、工人、管理
初期量太小 → 成本极高 → 不划算
等月销 500+ 台再考虑
```

## 创业公司特殊策略

### 策略 1 — 设计时考虑标准件

```
自定义零件（开模/定制）= 贵 + 慢 + 有起订量
标准件（市面现成有）= 便宜 + 快 + 随时能买

设计原则：
  能用标准件就不用定制件
  能用 M3/M4/M5 螺丝就不用特殊规格
  能用标准铝型材就不用开模铝件
  能用市售步进电机就不用定制电机

这叫 DFM for Supply Chain（面向供应链的设计）
```

### 策略 2 — 深圳生态系统

```
深圳华强北 — 电子元器件（全球最大电子市场）
  半小时能买齐所有电子零件
  当天就能拿到样品

深圳周边工厂
  PCB 打样：嘉立创，24 小时出货，5 片 ¥20
  铝合金加工：CNC 代加工，3-5 天出样品
  3D 打印：SLA/FDM，隔天出件
  开模：硅胶模 ¥500-2000，小批量够用

创业公司很多搬到深圳的原因：
  从设计到样品，别的城市要 2 周，深圳要 2 天
```

### 策略 3 — 用 3D 打印做非结构件

```
很多零件不需要金属，塑料就行：
  - 外壳 → 3D 打印
  - 传感器支架 → 3D 打印
  - 线缆固定夹 → 3D 打印
  - 旋钮/按钮 → 3D 打印

初期：3D 打印一件 ¥5-20，不需要开模
量产后：换成注塑，单件 ¥0.5-2

关键结构件（框架、导轨安装座）不要 3D 打印
→ 强度不够 → 用铝 CNC 代加工
```

### 策略 4 — 软件创业的硬件策略

```
AI CAM 公司的核心是软件：

  硬件策略：
    1. 支持市面上已有的机器（Shapeoko, 3018, X-Carve）
    2. 做一个推荐配置清单（买这些零件组装）
    3. 做好自己的控制板（软硬一体）
    4. 等用户量大了再自己做整机

  不需要一开始就自己造硬件：
    - 先做纯软件产品（SaaS / APP）
    - 用户用自己已有的机器
    - 验证 AI 参数推荐的价值
    - 有用户和收入后再做硬件
```

## 采购成本随量的变化

```
以滚珠丝杠 SFU1605-300mm 为例：

  淘宝零售:     ¥120/根   （买 1 根）
  1688 小批量:  ¥80/根    （买 20 根）
  工厂直采:     ¥55/根    （买 100 根）
  深度合作:     ¥40/根    （买 500+ 根，签年度框架）

量越大越便宜，但前提是有量
初期没有量 → 只能接受零售价 → 成本高 → 定价高 → 销量少 → 量上不去

这是硬件创业的鸡生蛋问题
解决方法：
  - 初期接受高成本，靠软件利润补贴
  - 先做软件服务，硬件利润不是重点
  - 等用户基数上来，再谈硬件降本
```
