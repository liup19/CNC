## 各渠道特点

### 淘宝 / 天猫 — 最常用

```
优势：
  - 中文沟通方便，可以定制尺寸
  - 发货快（国内 2-3 天）
  - 售后相对有保障
  - 可以买到整机套件

适合买的东西：
  - 铝型材（告诉商家切多长，打好孔）
  - 滚珠丝杠（告诉商家行程长度，配好螺母和轴承座）
  - 线性导轨（指定长度和型号，商家切割）
  - 刀具套装（10支装，各种直径，¥30-80）
  - 主轴电机（800W 水冷/风冷，¥150-400）

搜索关键词示例：
  "滚珠丝杠 SFU1204 300mm"  → 精确找到你需要的规格
  "ER11 夹头套装"            → 夹不同直径刀具的套件
  "线性导轨 MGN12H 300mm"    → 12mm 宽导轨，300mm 长
  "步进电机 NEMA23"          → 标准尺寸，57mm 方形
  "CNC 刀具套装 硬质合金"     → 一套各种尺寸的铣刀
```

### 1688（阿里巴巴批发）— 比淘宝便宜

```
优势：
  - 价格比淘宝低 20-40%
  - 可以联系工厂直接定制
  - 小批量也能买（很多支持 1 件起）

劣势：
  - 界面不如淘宝友好
  - 部分商家起批量大（5 件起）
  - 售后不如淘宝

适合买的东西：
  - 标准件（螺丝、螺母、轴承、联轴器）
  - 刀具（批量买更划算）
  - 电机、驱动器（量大价低）

技巧：
  - 搜到商品后跟客服说"买一件行不行"
  - 很多标着"10件起批"实际买 1 件也卖
  - 加一点运费就行
```

### AliExpress — 海外用户 / 价格对比

```
优势：
  - 很多中国工厂直营店
  - 某些组件比国内还便宜（工厂直接出口价）
  - 标准化零件型号清晰

劣势：
  - 发货慢（7-30 天）
  - 退换货麻烦
  - 国内用户没必要，除非淘宝找不到

适合海外用户购买：
  - 完整 CNC 套件（3018 改装件）
  - 步进电机驱动板（TMC2209 等）
  - 标准件（GT2 皮带、轮子、轴承）
```

### 闲鱼（二手）— 捡漏

```
适合买的东西：
  - 整机（3018 / Shapeoko / 自制机）
    很多人买来玩几次就出，价格打 3-5 折
    "3018 CNC" 闲鱼搜，¥200-400 就能买到

  - 主轴电机
    几乎不会坏，二手和新的一样用

  - 铝型材框架
    加工好的框架，买回来直接组装

不适合买的东西：
  - 刀具（消耗品，用了就磨损）
  - 皮带（老化拉伸）
  - 电子元件（可能有问题）
  - 轴承/导轨（磨损不可逆）

注意：
  - 让卖家拍视频确认机器能正常运行
  - 问清楚为什么出（升级了？吃灰了？有问题？）
  - 最好同城面交，能看到实物
```

## 按购买阶段组织

### 阶段 1 — 入门体验（预算 ¥500-1000）

```
方案：买一台 3018 整机
渠道：淘宝 ¥500-800 / 闲鱼 ¥200-400

包含：框架+电机+主轴+控制板+电源
买回来只需要装刀具、下软件就能用

再单独买：
  - 刀具套装（淘宝 ¥30-50，10 支装）
  - Z-Probe 探针（淘宝 ¥15-30）

理由：先跑通流程，确定自己需要什么
不推荐上来就买零件自己组装
```

### 阶段 2 — 升级改造（预算 ¥300-800）

```
在 3018 基础上换零件：

1. 丝杠换滚珠丝杠（淘宝）
   SFU1204 × 3 轴，含螺母+轴承座
   ¥60-120/轴 → 三轴 ¥180-360

2. 加刚性（淘宝铝型材）
   2040 铝型材加固
   ¥50-100

3. 换更好的主轴（淘宝）
   500W/800W 主轴 + 变频器
   ¥150-400

4. 换控制板（淘宝）
   GRBL ESP32 或 FluidNC
   ¥30-80

渠道：全部淘宝，标准件 1688 更便宜
```

```
从零组装一台比 Shapeoko 更好的机器：
```

## 省钱技巧

```
1. 刀具 — 买套装不买单支
   单支 ¥10-20，套装 10 支 ¥30-50
   淘宝搜 "CNC 刀具套装 硬质合金"

2. 标准件 — 1688 比 淘宝便宜
   螺丝、螺母、轴承、联轴器
   加购物车一起买，省运费

3. 定制件 — 给商家图纸
   铝型材、丝杠、导轨都可以定制长度
   发 CAD 图纸给客服 → 报价 → 生产
   不要买标准长度自己切（切不好，精度差）

4. 电机驱动 — TMC2209 适合消费级
   ¥5-8/片（1688），静音 + 防丢步
   淘宝模块板 ¥15-25/块
   不需要买贵的 DM542（工业级，用不上）

5. 闲鱼蹲整机
   很多人 3018 玩两个月就出
   ¥200-400 拿下全套 → 改装升级 → 比买新的省一半
```
