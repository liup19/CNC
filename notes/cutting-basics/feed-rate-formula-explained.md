# 进给率公式详解：Feed = RPM × z × fz

> 最后更新：2025-05-15

---

## 公式

```
Feed = RPM × z × fz
```

## 每个符号的含义

### RPM — 主轴转速（转/分钟）

刀每分钟转多少圈。

```
RPM = 10610

想象一下：
  1秒钟转 10610÷60 ≈ 177 圈
  每转一圈用时 ≈ 0.0056 秒
```

RPM 由切削速度 (Vc) 和刀具直径 (D) 决定：

```
RPM = (Vc × 1000) / (π × D)

Vc = 切削速度，单位 m/min（材料属性，查表得到）
D  = 刀具直径，单位 mm（刀具的物理尺寸）
π  = 3.14159
```

---

### z — 刃数（齿数）

刀具有几个切削刃（flutes）。

```
z = 2（示例中是 2 刃刀具）

常见刀具的刃数：

  1刃（单槽）:  z=1
  ████████░░  一个刃，排屑空间大
  用途：切铝（切屑软而粘，需要大排屑空间）

  2刃（双槽）:  z=2
  ████░░████  两个刃，最常见
  用途：铝、木材、塑料（通用型）

  3刃（三槽）:  z=3
  ███░███░███  三个刃，较光滑
  用途：精加工、钢材粗加工

  4刃（四槽）:  z=4
  ██░██░██░██  四个刃，表面好
  用途：钢材精加工
```

**刃数的影响：** 刃数越多，每转一圈切的次数越多，表面越光滑，但排屑空间越小，容易堵屑。

```
z=2 时：Feed = 10610 × 2 × 0.03 = 637 mm/min
z=4 时：Feed = 10610 × 4 × 0.03 = 1273 mm/min  ← 同样 fz，Feed 翻倍

但 z=4 在铝材中容易堵屑 → 实际上 z=2 更适合铝
```

---

### fz — 每刃进给量（切屑负载, Chip Load）

**每个刀刃转一圈时切入材料的厚度。**

```
fz = 0.03 mm/tooth

单位：mm/tooth（毫米/每齿）

直观理解：
  fz = 0.03mm ≈ 一张 A4 纸厚度的 1/3
  fz = 0.10mm ≈ 一根头发丝的粗细
```

**为什么 fz 是最关键的参数：**

```
fz 太大 (>0.10mm for 6mm刀):
  刀刃切入太深 → 切削力过大 → 断刀

fz 太小 (<0.01mm):
  刀刃切不进去 → 在材料上摩擦 → 发热 → 烧刀/烧材料

fz 刚好 (0.03-0.08mm for 6mm刀切铝):
  刀刃干净地切下切屑 → 切削力合理 → 效率与安全的平衡
```

**fz 从哪里来：**

```
fz 取决于三个因素：

1. 刀具直径 D:
   D=3mm  → fz = 0.01-0.04 mm   ← 小刀，切薄一点
   D=6mm  → fz = 0.03-0.08 mm   ← 中等
   D=12mm → fz = 0.05-0.12 mm   ← 大刀，可以切厚一点

2. 材料类型:
   铝合金:  fz × 1.0（基准）
   硬木:    fz × 1.2（木材好切）
   MDF:     fz × 1.5（更软）
   钢铁:    fz × 0.3（难切，必须薄）

3. 刀具材质:
   硬质合金: fz × 1.0（基准）
   HSS:     fz × 0.6（高速钢不如硬质合金硬）
```

---

### Feed — 进给率（最终结果）

**刀具在材料上每分钟移动的距离。**

```
Feed = RPM × z × fz
     = 10610 × 2 × 0.03
     = 637 mm/min

意思是：刀具以每分钟 637 毫米的速度在材料上平移
      ≈ 每秒移动 10.6 毫米
      ≈ 每秒大约 1 厘米
```

**公式的物理含义拆解：**

```
RPM × z = 每分钟有多少个刀刃经过材料

  RPM = 10610 转/分钟
  z = 2 刃
  → 每分钟有 10610 × 2 = 21220 个刀刃经过材料

RPM × z × fz = 每分钟刀刃总共切出多长

  21220 个刃 × 每刃切 0.03mm
  = 21220 × 0.03
  = 636.6 mm/min
  ≈ 637 mm/min

  这就是刀具需要前进的速度
```

**用生活比喻理解：**

```
想象一个旋转的披萨切割轮在面皮上切直线：

  RPM = 轮子每分钟转多少圈
  z   = 轮子上有几个刀片
  fz  = 每个刀片转一圈时切入面皮多深
  Feed= 轮子在面皮上每分钟滚多远

  转得快 + 刀片多 + 每次切得深 → 轮子需要滚得快
  转得慢 + 刀片少 + 每次切得浅 → 轮子滚得慢
```

## 反过来算：已知 Feed 推算 fz

实际使用中，经常需要反向计算 — 验证给定参数是否安全：

```
fz = Feed / (RPM × z)

示例：用户设置 Feed=800, RPM=10000, z=2
  → fz = 800 / (10000 × 2) = 0.04 mm/tooth ✓ 在安全范围

示例：用户设置 Feed=2000, RPM=10000, z=2
  → fz = 2000 / (10000 × 2) = 0.10 mm/tooth ⚠️ 偏高，断刀风险

示例：用户设置 Feed=50, RPM=10000, z=2
  → fz = 50 / (10000 × 2) = 0.0025 mm/tooth ✗ 太小！摩擦非切削
```

## 参数联动的实际场景

### 场景 1：RPM 被机器限制

```
计算得 RPM=15915，但机器最高只有 12000

解决：RPM clamp 到 12000
  新 Feed = 12000 × 2 × 0.03 = 720 mm/min

  fz 不变（0.03），只是整体速度降低
```

### 场景 2：想提高加工速度

```
当前: Feed=637, 切深=1.5mm
目标: Feed=1000

方法 1 — 提高 fz:
  1000 = 10610 × 2 × fz → fz = 0.047（从 0.03 提到 0.047）
  风险：fz 偏高，切削力增大

方法 2 — 换 4 刃刀:
  1000 = 10610 × 4 × fz → fz = 0.024
  fz 反而降低了，更安全
  但 4 刃刀切铝容易堵屑 → 可能不适合

方法 3 — 降低切深（保持 Feed 不变，多走几刀）:
  Feed 保持 637，切深降到 1.0mm
  多走 50% 的刀路，但每刀更安全

  实际工程中最常用的是方法 3
```

### 场景 3：换材料

```
从铝换到钢，同一个 6mm 2刃刀：

铝: fz=0.03 → Feed = 10610 × 2 × 0.03 = 637 mm/min
钢: fz=0.01 → Feed = 5305 × 2 × 0.01  = 106 mm/min

钢的 Feed 只有铝的 1/6，原因：
  - fz 从 0.03 降到 0.01（钢硬 3 倍，切薄一些）
  - RPM 从 10610 降到 5305（Vc 也低，刀不能转太快）
  两个因素叠加 → Feed 大幅降低
```
