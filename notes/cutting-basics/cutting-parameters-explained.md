# CNC 切削参数详解 — 所有符号逐个解释

> 最后更新：2026-05-16

---

## 参数总览

本文涉及的所有公式和符号：

```
公式 1: RPM = (Vc × 1000) / (π × D)
公式 2: Feed = RPM × z × fz
公式 3: 最大切深 ap_max = 刚性系数 × D
公式 4: Stepover = D × 百分比%
公式 5: Fc = Kc × ap × ae × fz
```

## 符号速查表

| 符号 | 名称 | 单位 | 来自哪里 | 含义一句话 |
|------|------|------|----------|-----------|
| RPM | 主轴转速 | 转/分钟 (rev/min) | 公式1计算 | 刀每分钟转多少圈 |
| Vc | 切削速度 | 米/分钟 (m/min) | 查材料表 | 刀刃在材料表面滑动的线速度 |

| Fc | 切削力 | 牛顿 (N) | 公式5计算 | 刀具切削时受到的阻力 |
| Kc | 比切削力 | 牛顿/平方毫米 (N/mm²) | 查材料表 | 切掉单位面积材料需要多大的力 |

## 1. RPM — 主轴转速

```
符号: RPM (Revolutions Per Minute)
中文名: 主轴转速、每分钟转数
单位: 转/分钟 (rev/min 或 rpm)
```

**含义：** 主轴（装刀具的那个旋转轴）每分钟转多少圈。

**直观理解：**

```
RPM = 10610 时：

  每秒转: 10610 ÷ 60 = 177 圈/秒
  每转一圈: 1 ÷ 177 = 0.0056 秒 = 5.6 毫秒

  快到肉眼看不到，只能看到一个圆盘
```

**RPM 不是随便设的，由公式 1 计算得到：**

```
RPM = (Vc × 1000) / (π × D)
```

**在公式 2 中的作用：** RPM 是进给率计算的基础。转得越快，刀刃经过材料的次数越多，刀具就可以走得越快。

---

## 2. Vc — 切削速度

```
符号: Vc (Cutting Speed / Surface Speed)
中文名: 切削速度、线速度
单位: 米/分钟 (m/min)
```

**含义：** 刀刃在材料表面"划过"的线速度。不是转速，而是刀刃边缘的实际运动速度。

**直观理解：**

```
想象一个轮子在地面上滚动：

  轮子边缘的线速度 = 轮子的转速 × 轮子的周长

  对 CNC 刀具来说：
  刀刃的线速度 (Vc) = 刀的周长 × 转速

  Vc 本质上是"刀刃划过材料有多快"
```

**Vc 是材料属性，查表得到，不是自己设的：**

| 材料 | Vc 范围 — 硬质合金刀 (m/min) | 说明 |
|------|------------------------------|------|
| 6061 铝合金 | 300–500 | 软，可以快 |
| 黄铜 | 200–400 | 较软 |
| 亚克力 | 150–300 | 中等 |
| 软钢 (1045) | 100–200 | 硬，要慢 |
| 不锈钢 (304) | 80–150 | 很硬，更慢 |
| MDF | 300–600 | 极软 |
| 硬木 | 200–400 | 较软 |

**Vc 还受刀具材质影响：**

```
同一个材料 6061 铝：

  硬质合金刀 (Carbide):   Vc = 300–500 m/min  ← 硬，可以快
  高速钢刀 (HSS):          Vc = 120–200 m/min  ← 软一些，要慢
  HSS × 0.4 ≈ Carbide 的 40%

  所以刀具材质直接影响 RPM 计算
```

**在公式 1 中的作用：** Vc 是 RPM 计算的起点。给定 Vc 和 D，算出 RPM。

## 3. D — 刀具直径

```
符号: D (Diameter)
中文名: 刀具直径
单位: 毫米 (mm)
```

**含义：** 刀具切削部分的直径，即刀能切出的最宽槽的宽度。

**直观理解：**

```
  俯视图：

       ↓ D = 6mm
  ┌─────────┐
  │  ○○○○○  │  ← 刀具截面（圆）
  │  ○○○○○  │
  │  ○○○○○  │
  └─────────┘

  D=6mm 的刀能切出 6mm 宽的槽
  D=3mm 的刀能切出 3mm 宽的槽
  D=12mm 的刀能切出 12mm 宽的槽
```

**消费级常见刀具直径：**

| 直径 | 用途 | ER11 夹头支持 |
|------|------|--------------|
| 1mm | 精细雕刻 | ✓ |
| 3mm (1/8") | 细活、小特征 | ✓ |
| 6mm (1/4") | 通用，最常用 | ✓ |
| 6.35mm (1/4") | 通用（英制） | ✓（刚好） |
| 10mm | 粗加工去量大 | ✗（超出 ER11） |

**D 在多个公式中反复出现：**

```
公式 1: RPM = (Vc×1000)/(π×D)     ← D 越大，RPM 越低
公式 3: ap_max = 系数 × D          ← D 越大，允许切越深
公式 4: Stepover = D × 百分比       ← 步距按 D 的比例算
公式 5: Fc 间接影响                ← D 影响可用 fz 范围
```

**关键洞察：** D 是连接所有参数的核心变量。几乎所有参数都和 D 有关系。

## 4. π — 圆周率

```
符号: π (Pi)
数值: 3.14159...
单位: 无单位（无量纲常数）
```

**含义：** 圆的周长与直径之比。出现在公式 1 中是因为要把线速度 (Vc) 转换为转速 (RPM)。

**为什么公式 1 里有 π：**

```
圆的周长 C = π × D

刀转一圈，刀刃走过的距离 = π × D

所以：
  刀刃每分钟走过的距离 = π × D × RPM

  这个距离就等于切削速度 Vc（单位换算后）：
  Vc (m/min) = π × D (mm) × RPM / 1000

  反过来：
  RPM = (Vc × 1000) / (π × D)

  1000 是单位换算：Vc 用 m/min，D 用 mm
  1m = 1000mm，所以 Vc×1000 把米换算成毫米
```

## 5. z — 刃数（齿数）

```
符号: z (Number of Flutes / Number of Teeth)
中文名: 刃数、齿数、槽数
单位: 个（无量纲整数）
```

**含义：** 刀具圆周上有几个切削刃。每个切削刃就是一条可以切材料的"刀口"。

**直观理解：**

```
从刀具底部往上看（截面图）：

  1刃 (z=1):          2刃 (z=2):          4刃 (z=4):
     ╱                 ╱ ╲                ╱ ╲ ╱ ╲
    ╱  一个刀口        ╱   ╲              │   │   │
   ╱                   ╱     ╲            │   │   │
                      ╱       ╲           ╲   ╱   ╱
  排屑空间最大        两个对称刃           四个刃
  切铝首选            最通用              钢材精加工
```

**刃数对加工的影响：**

```
z=1: 每转只有 1 个刃切材料
     → 排屑空间大（切屑容易排出）
     → 适合粘性材料（铝：切屑软而粘，需要大空间排屑）
     → Feed 较低（因为只有 1 个刃在工作）

z=2: 每转有 2 个刃切材料
     → 排屑空间和切削效率的平衡
     → 消费级最通用的选择
     → Feed 是 z=1 的 2 倍

z=4: 每转有 4 个刃切材料
     → 切削频率高，表面更光滑
     → 排屑空间小（4 个刃占了很多位置）
     → 适合钢材（切屑短而碎，不需要大排屑空间）
     → Feed 是 z=2 的 2 倍
```

**在公式 2 中的作用：**

```
Feed = RPM × z × fz

z 越大 → 同样 RPM 下，Feed 越高（走得越快）

示例 RPM=10000, fz=0.03:
  z=1: Feed = 10000 × 1 × 0.03 = 300 mm/min
  z=2: Feed = 10000 × 2 × 0.03 = 600 mm/min  ← 翻倍
  z=4: Feed = 10000 × 4 × 0.03 = 1200 mm/min ← 再翻倍
```

**消费级经验：**

| 材料 | 推荐 z | 理由 |
|------|--------|------|
| 铝合金 | 1–2 | 切屑粘，需要大排屑空间 |
| 木材/MDF | 1–2 | 切屑量大，需要排屑 |
| 塑料/亚克力 | 1–2 | 切屑容易融化粘刀 |
| 钢铁 | 3–4 | 切屑短碎，可以用多刃提高效率 |

## 6. fz — 每刃进给量（切屑负载, Chip Load）

```
符号: fz (Feed per Tooth / Chip Load)
中文名: 每刃进给量、切屑负载、每齿进给
单位: 毫米/齿 (mm/tooth)
```

**含义：** 单个刀刃转一圈时，刀具沿进给方向前进的距离。就是"每个齿咬下去多厚"。

**这是整个切削参数系统中最重要的参数。**

**fz 的运动方向 — 沿进给方向（刀具平移方向）：**

fz 不是往下切的，也不是往横切的，而是**沿着刀具走动的方向**切进去的厚度。

```
三维视角 — ap、ae、fz 分别在三个互相垂直的方向：

         Z轴（向下）
         ↓ ap（轴向切深，竖着吃多深）
         │
         │
    ┌────┼────┐
    │ 刀 │    │
    │ 具 │    │
    │ ██ │    │
    │ ██ │    │
    └────┼────┘
    ╔════╪════╗
    ║ ←ae→   ║ ← ae（径向切宽，横着切多宽）
    ║         ║    垂直于进给方向，也垂直于 Z轴
    ║    fz→  ║ ← fz（切屑厚度，沿进给方向咬多厚）
    ╚═════════╝    刀具实际运动的方向

  三个方向完全独立、互相垂直：
    ap = Z 轴方向（竖着，上下）
    ae = 刀具径向（横着，垂直于进给和 Z轴）
    fz = 进给方向（刀具平移的方向）
```

**用切面包的类比理解三个方向：**

```
场景：一块大面包，你用刀切下一些面包片。

面包块（侧视图）：

  ┌──────────────────────────────┐
  │                              │
  │          面包块               │
  │                              │
  │                              │
  └──────────────────────────────┘


步骤 1 — ap（深度）= 刀往下压多深

  你把刀放在面包上，往下压：

         刀
      ┌──╋──┐
      │  ║  │
      │  ║  │ ← ap（往下压了多深）
      │  ║  │
      └──╋──┘
  ╔═══╬══════════════════════════╗
  ║   ║                          ║
  ║   ║↓ ap                      ║
  ║   ║                          ║
  ║   ║        面包块             ║
  ╚══════════════════════════════╝

  ap 大 → 刀压得深 → 切入面包更多
  ap 小 → 刀压得浅 → 只切到表面

  方向：竖直向下（Z 轴）


步骤 2 — ae（宽度）= 刀刃接触到面包多长

  刀有一定的长度，接触面包的范围就是 ae：

      刀刃
  ════════
  ╔═╗
  ║█║ ← ae（刀刃接触面包的范围）
  ║█║
  ╚═╝
  面包表面

  ae 大 → 刀刃接触面包范围大 → 切得宽
  ae 小 → 刀刃接触面包范围小 → 切得窄

  方向：水平，垂直于进给方向


步骤 3 — fz（厚度）= 每次推刀切下来的面包片多厚

  你往前推刀，切下来一片：

  推刀方向 →→→

  你往前推了 fz 的距离，切下来一片面包：

              ← fz（推刀的距离 = 切片的厚度）→
            ┌─────────────────────────────────┐
            │           切下来的面包片          │
            │           厚度 = fz              │
            └─────────────────────────────────┘
  ╔══════════════════════════════════════════════╗
  ║                 剩下的面包                    ║
  ╚══════════════════════════════════════════════╝

  fz 大 → 推刀远 → 切片厚
  fz 小 → 推刀近 → 切片薄

  方向：水平，进给方向（推刀方向）
```

```
三个动作完全独立 — 你可以自由组合：

  "薄薄地、浅浅地、窄窄地切一小片"
  ap=小 ae=小 fz=小
  → 切下来一小块薄的面包片

  ┌──┐
  │  │ ← 薄片
  └──┘

  "深深地、宽宽地、厚厚地切一大块"
  ap=大 ae=大 fz=大
  → 切下来一大块厚的面包片

  ┌──────────────┐
  │              │
  │              │
  │   大块       │
  │              │
  └──────────────┘

  "深但窄又薄"
  ap=大 ae=小 fz=小
  → 切得深但每次只切一小条

  ┌──┐
  │  │
  │  │
  │  │ ← 又深又窄又薄的一条
  │  │
  └──┘

  切下来的面包片体积 = ap × ae × fz
  切下来的面积越大/越厚 → 费的力气越大
  这就是为什么 Fc = Kc × ap × ae × fz
```

**fz 是怎么产生的：**

```
刀具在旋转的同时沿进给方向平移。

  时间 t0: 刀刃在位置 A
  时间 t1: 刀刃转了一圈，回到了同样的角度
           但刀具已经往前移动了 fz 的距离

  侧视图 — 刀刃的运动轨迹：

    刀刃旋转 →↗→↗→↗→
              切屑
    ┌───┐   ╔═══╗
    │刀 │ → ║███║ ← fz（刀刃前进的距离）
    │具 │   ╚═══╝    就是切屑的厚度
    └───┘
    ──────────────── 材料

  fz = 刀具每转一圈，沿着走动方向前进的距离 ÷ 齿数 z

  Feed = RPM × z × fz
  fz = Feed / (RPM × z) = 每分钟前进距离 / 每分钟总刃次
     = 每个刃分摊的前进距离 = 每个刃咬下来的切屑厚度
```

**直观理解 fz 的大小：**

```
fz = 0.03 mm/tooth

一刃下去切 0.03mm 厚的切屑：
  A4 纸厚度 ≈ 0.1mm
  0.03mm ≈ A4 纸厚度的 1/3

fz = 0.10 mm/tooth
  一根头发丝的直径 ≈ 0.08-0.10mm
  0.10mm ≈ 一根头发丝
```

**为什么 fz 最关键：**

```
fz 直接决定：

1. 切削力
   fz 大 → 每个刃切得多 → 切削力大 → 可能断刀
   fz 小 → 每个刃切得少 → 切削力小

2. 表面质量
   fz 大 → 刀痕深 → 表面粗糙
   fz 小 → 刀痕浅 → 表面光滑

3. 刀具寿命
   fz 太小 → 刀刃在材料上摩擦而非切削 → 发热 → 烧刀
   fz 合理 → 干净地切下切屑 → 刀具正常磨损

4. 安全性
   fz 太大 → 切削力超过刀具强度 → 断刀飞出
```

**fz 的安全范围：**

```
6mm 硬质合金平底刀的 fz 范围：

  0.01  0.03      0.06      0.08  0.10  mm/tooth
   │     │         │         │     │
   危险  ┃──推荐──┃──可用──┃  危险
   摩擦  安全范围           断刀风险

  fz < 0.01: 刀刃不是在切，是在磨 → 发热烧刀
  fz > 0.10: 刀具承受不住 → 断刀
```

**fz 受什么影响：**

```
1. 刀具直径 D（最主要因素）:
   D=1mm  → fz = 0.001–0.010  小刀必须切薄
   D=3mm  → fz = 0.010–0.040
   D=6mm  → fz = 0.030–0.080
   D=12mm → fz = 0.050–0.120  大刀可以切厚

   经验公式: fz_推荐 ≈ 0.01 × D^(0.5)
   D=6mm → fz ≈ 0.01 × 2.45 = 0.0245 (偏保守的基线)

2. 材料硬度:
   铝合金: fz × 1.0（基准）
   硬木:   fz × 1.2（木材软，可以多切一点）
   MDF:    fz × 1.5（很软）
   钢铁:   fz × 0.3（硬，必须切薄）
   不锈钢: fz × 0.2（更硬更粘）

3. 刀具材质:
   硬质合金:  fz × 1.0
   HSS 高速钢: fz × 0.6（不如硬质合金硬）
   涂层硬质合金: fz × 1.1（涂层增加耐磨性）

4. 刚性等级（间接影响，通过安全约束）:
   VERY_LOW: fz × 0.6
   LOW:      fz × 0.8
   MEDIUM:   fz × 1.0
   HIGH:     fz × 1.0
```

**在公式 2 中的作用：** fz 是 Feed 计算的核心输入。

**在公式 5 中的作用：** fz 直接影响切削力大小。

## 7. Feed — 进给率

```
符号: Feed (Feed Rate)
中文名: 进给率、进给速度
单位: 毫米/分钟 (mm/min)
```

**含义：** 刀具在材料上每分钟移动的距离。这就是 CAM 软件里要设的"F 值"（G-code 中的 F 指令）。

**计算公式：**

```
Feed = RPM × z × fz

物理含义拆解：
  RPM × z = 每分钟有多少个刀刃经过材料（刃次/分钟）
  RPM × z × fz = 每分钟所有刀刃总共切出多长的材料
  这个长度就是刀具需要前进的速度

示例:
  RPM=10610, z=2, fz=0.03
  每分钟刃次 = 10610 × 2 = 21220 次
  每次切 0.03mm
  总长 = 21220 × 0.03 = 636.6 ≈ 637 mm/min
```

**实际体感：**

```
Feed 的快慢感知：

  100 mm/min ≈ 每秒 1.7mm  ← 肉眼可见的慢，像蜗牛爬
  637 mm/min ≈ 每秒 10.6mm ← 中等，正常铝材加工
  2000 mm/min ≈ 每秒 33mm  ← 快，木材粗加工
  5000 mm/min ≈ 每秒 83mm  ← 很快，需要工业级机器
```

**G-code 中的表示：**

```gcode
G1 X50 Y50 F637
              ↑
         F 值 = Feed = 637 mm/min
```

**反向计算（验证参数是否安全）：**

```
fz = Feed / (RPM × z)

用户设 Feed=800, RPM=10000, z=2:
  fz = 800 / (10000 × 2) = 0.04 mm/tooth ✓ 安全

用户设 Feed=2000, RPM=10000, z=2:
  fz = 2000 / (10000 × 2) = 0.10 mm/tooth ⚠ 偏高

用户设 Feed=50, RPM=10000, z=2:
  fz = 50 / (10000 × 2) = 0.0025 mm/tooth ✗ 太小，会烧刀
```

## 8. ap — 轴向切深（Depth of Cut）

```
符号: ap (Axial Depth of Cut)
中文名: 轴向切深、切深、吃刀深度
单位: 毫米 (mm)
```

**含义：** 刀具底部低于材料表面的距离。即刀具嵌入材料有多深。

**重要：ap 不是"切削动作"，是"刀具位置"。**

```
ap 不是刀往下切出来的，而是刀具定位在材料表面以下 ap 的深度，
然后水平移动时，侧面刀刃把嵌入部分的材料刮掉。

  误区：以为 ap 是刀往下"切"出来的
  ✗ 刀像钉子一样往下扎？不是！

  实际：ap 是刀具嵌入材料的深度（位置关系）
  ✓ 刀先降到 Z=-1.5mm，然后水平移动，侧面刀刃刮掉材料
```

### 8.1 微观切削过程

**切削实际上只在水平方向发生：**

```
正视刀具侧面（沿刀具轴线方向看）：

  刀具截面         材料

  ┌─────┐      ┌──────────────┐
  │     │      │              │
  │  刀 │ →→→ │  ▓▓▓▓▓▓▓▓▓▓ │
  │  具 │ 水平 │  ▓▓▓▓▓▓▓▓▓▓ │
  │     │ 移动 │  ▓▓▓▓▓▓▓▓▓▓ │
  └─────┘      └──────────────┘

  放大看刀刃接触材料的地方：

  ┌────┐
  │刀  │    ╔═══╗ ← fz（每齿削下这么厚）
  │具  │ →  ║切屑║   沿水平方向
  │侧  │    ╚═══╝
  │面  │    ═══════════════
  │    │    ▓▓▓▓▓▓▓▓▓▓▓▓▓▓ ← 剩下的材料
  └────┘

  ↑ ae（刀具侧面嵌入材料的宽度）

  注意：切削只在水平方向发生！
  刀刃把材料水平方向削掉 fz 厚的一层
```

**ap 在微观里是什么 — 刀具嵌入材料的深度：**

```
侧视图（从刀具侧面看）：

       Z=0 材料表面
       ═════════════════
  ┌────┤             ┌────┐
  │    │             │    │
  │ 刀 │  ← ae →    │ 刀 │
  │ 具 │             │ 具 │
  │    │             │    │
  └────┤             └────┤
  ▓▓▓▓▓╪═════════════╪▓▓▓▓
  ▓▓▓▓▓╪▓▓▓▓▓▓▓▓▓▓▓▓╪▓▓▓▓
  ▓▓▓▓▓╪▓▓▓▓▓▓▓▓▓▓▓▓╪▓▓▓▓
       ↑              ↑
       │   ← ap →     │
       │  刀具嵌入深度  │
       Z=-1.5mm

  ap = 刀具嵌入多深（定位，不是切削动作）
  ae = 刀具嵌入多宽（定位，不是切削动作）
  fz = 每齿实际削掉的厚度（唯一的切削量）
```

**为什么硬材料也能切 — 因为是"刮"不是"压"：**

```
刀刃的微观视角：

  刀刃是一个极小的楔形，比大多数材料都硬（硬质合金 ~HRA90）

  ┌─┐
  │ ╲  ← 刀刃（硬质合金，非常锋利）
  │  ╲
  │   ╲→ 切削方向（旋转产生的速度）
  └───┘
  ▓▓▓▓▓ ← 材料

  刀刃不是"压"进去的，是"刮"过去的：
  刀刃以极高速度掠过材料表面 → 把材料剪切下来

  类比：
    削苹果皮 — 刀在苹果表面滑动，削下一层皮
    刨木头   — 刨刀划过去，削下木花

  材料硬度影响的是"每层敢削多少"：
    硬材料 → 刀刃刮过时阻力大 → 切削力大
    → 如果 ap 太大 → 切削力超过机器承受 → 机器抖动/断刀
    → 所以必须减小 ap → 每层少削一点 → 多走几层
```

### 8.2 逐层切削的完整过程

```
目标：在铝块上铣一个 4.5mm 深的槽，ap=1.5mm

Layer 1: 刀具降到 Z=-1.5（ap=1.5mm）

  ┌──┐
  │刀│ →→→→→→→→→→→→ 沿槽的形状走完整圈
  └──┘                （刀具旋转了上万圈，但只在这一层水平移动）
  ══════════════════
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓

  走完后：材料表面被削掉了 1.5mm
  ════╗         ╔════
  ▓▓▓▓║  槽 1.5mm深 ║▓▓▓▓
  ▓▓▓▓╚═════════╝▓▓▓▓
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓

Layer 2: 刀具降到 Z=-3.0（再 ap=1.5mm）

  ┌──┐
  │刀│ →→→→→→→→→→→→ 再沿形状走完整圈
  └──┘
  ════╗         ╔════
  ▓▓▓▓║         ║▓▓▓▓
  ▓▓▓▓╚═════════╝▓▓▓▓  ← 第2层削掉
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓

  走完后：槽深 3.0mm

Layer 3: 刀具降到 Z=-4.5（再 ap=1.5mm）
  同上

  最终：槽深 4.5mm（3层 × 1.5mm）
```

### 8.3 刀具是怎么进入材料的

**核心问题：材料是硬的，刀具怎么"降低到负 Z"？**

```
答案是：刀具底部也有切削刃（端刃），旋转时配合 Z 轴电机向下推，
像钻头一样切进去。

刀具底面（仰视图）：

     ╲ 刃１  ╱
      ╲     ╱
       ╲   ╱
        ╲ ╱
         ×  ← 中心
        ╱ ╲
       ╱   ╲
      ╱     ╲
     ╱  刃２  ╲

平底铣刀底部有 2 条端刃（z=2 时）
旋转时，端刃像钻头一样从底部切掉材料
```

```
直插下刀（Plunge）过程 — 端刃如何切入材料：

  刀具旋转方向 →→→

  Z=0（材料表面）
  ┌─────────────────┐
  │                 │   刀具从上方靠近
  │      ↓ 刀       │   端刃接触表面
  │   ↗ ╱╲ ↘      │   旋转 + 下压
  │  →  ╳  ←      │   每转一圈，端刃切掉
  │   ↘ ╲╱ ↙      │   一层薄薄的材料（厚度 = fz）
  │      ↓         │
  └─────────────────┘

  Z=-0.03mm  ← 转一圈后，切掉了 fz 厚的一层
  Z=-0.06mm  ← 再转一圈，又切掉一层
  ...
  Z=-1.5mm   ← 转 50 圈后，到达目标深度

  关键：不是"硬压进去"，而是"旋转着一点一点刮进去"
  每次只刮掉 0.03mm（一张纸的 1/3），需要的力很小
```

```
为什么旋转的刀能"切"进硬材料？

对比：

  ✗ 静止的圆棒压入铝块
    → 需要巨大力量（铝的屈服强度 ~276 MPa）
    → 只会压出一个坑，不是切削

  ✓ 旋转的锋利刀刃
    → 刃口极尖（刃口半径 ~5μm，比头发细 20 倍）
    → 集中应力 >> 材料强度 → 材料断裂分离
    → 切掉的那片（切屑）很薄（fz=0.03mm）
    → 需要的力很小（~几十牛顿，一只手就能按出来）
```

**三种下刀方式：**

```
方式1 — 直插下刀（Plunge）：最简单但最伤刀

  ┌──┐
  │刀│
  │具│  ↓↓↓ 直上直下
  └──┘
  ▓▓▓▓▓▓▓▓

  只有底部端刃在工作，排屑困难，发热大
  只用于软材料或浅深度


方式2 — 斜插下刀（Ramp）：边走边下，最常用

  侧视图：
  ┌──┐
  │刀│  ╲
  │具│    ╲  ← 刀具斜着进去（Z 下降的同时 XY 移动）
  └──┘      ╲
  ▓▓▓▓▓▓▓▓▓▓▓╲▓▓
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓

  Z 下降和 XY 移动同时进行
  侧面刀刃参与切削，排屑好，力小
  斜插角度通常 1-3°（很缓）


方式3 — 螺旋下刀（Helical）：挖槽时用

  刀具走螺旋线 — 一边转圈一边往下：

  从侧面看：
      高
      │  ╭─╮
      │ ╱   ╲
      │╱     ╲╮
      ╲      ╱╲
       ╲   ╱   ╲
        ╲ ╱     ╲
         ╳       ╲
        ╱╲        │
       ╱  ╲      │
      ╱    ╲     │
     ╱      ╲    │
                    低
    一圈一圈，每圈下降一点

  从正上方看（俯视）：
       ╭───╮
      ╱     ╲
     │  ←──╮ │
     │      │ │   就是一个圆
     │  ╰──→ │ │   但刀在逐渐往下走
      ╲     ╱
       ╰───╯

  生活中的例子：
    弹簧的铁丝 — 一圈圈缠绕
    螺丝钉的螺纹
    停车场的螺旋坡道

  CNC 里怎么用：
    第 1 圈：Z = 0 → -0.5mm
    第 2 圈：Z = -0.5 → -1.0mm
    第 3 圈：Z = -1.0 → -1.5mm  ← 到达目标深度

    效果：在实心材料里"钻"出一个和刀具一样大的圆洞
    侧面刀刃干活的比重最大 → 切削力均匀，最适合硬材料和深槽
```

```
三种方式对比：

  直插：端刃干活 → 发热大、排屑差 → 仅软材料/浅深
  斜插：侧面+端刃 → 均衡 → 最常用
  螺旋：侧面为主 → 最平稳 → 挖槽首选
```

### 8.4 ap 的上限

```
ap_max = min(刚性系数 × D, 刀具有效切削长度)

因素 1 — 机器刚性：
  刚性越好，允许嵌入越深（相对于刀具直径 D 的比例）

              铝      硬木     MDF     钢      塑料
  VERY_LOW:  0.10D   0.25D   0.30D   0.03D   0.15D
  LOW:       0.15D   0.35D   0.40D   0.05D   0.20D
  MEDIUM:    0.25D   0.50D   0.60D   0.08D   0.30D
  HIGH:      0.40D   0.75D   1.00D   0.15D   0.40D

  D=6mm 切铝:
    VERY_LOW: ap_max = 0.10 × 6 = 0.6mm   → 要走 8 层才能切到 5mm
    MEDIUM:   ap_max = 0.25 × 6 = 1.5mm   → 要走 4 层
    HIGH:     ap_max = 0.40 × 6 = 2.4mm   → 要走 3 层

因素 2 — 刀具切削长度：
  刀刃只有那么长，不能嵌入得比刀刃还深
  刀具规格: D=6mm, 切削长度=15mm
  → ap 不能超过 15mm（但刚性限制通常先生效）
```

**在公式 5 中的作用：** ap 直接影响切削力。ap 翻倍 → 切削力翻倍。因为嵌入更深 → 刀刃接触材料更多 → 需要更大的力。

### 8.5 ap 和 fz 的区别总结

```
ap = 每层总深度（和旋转圈数无关，是 Z 轴定位）
fz = 每转一圈，每个齿削下来的厚度（是实际切削量）

  ap 管的是：整个刀路（走几分钟）在 Z 方向嵌入多深
  刀具旋转管的是：在这几分钟里，刀刃高速刮过材料表面

  一次完整的层切削：

  刀具定位到 Z=-1.5mm（ap=1.5）
  → 刀具开始水平移动
  → 旋转的刀刃削下 fz=0.03mm 厚的切屑
  → 转了上万圈，每圈每个齿削 0.03mm
  → 走完整层刀路 → 这一层 ap=1.5mm 深的材料全部削完
  → 刀具下降到 Z=-3.0mm → 走下一层
```

## 9. ae — 径向切宽（Radial Depth of Cut / Stepover）

```
符号: ae (Radial Depth of Cut)
中文名: 径向切宽、径向切深
单位: 毫米 (mm)
```

**含义：** 刀具沿径向（垂直于刀具轴线的方向）切入材料的宽度。实际上就是**步距（Stepover）**的绝对值。

**直观理解：**

```
俯视图 — 刀具从左往右走两刀：

  ae = 步距 = 3mm (50% D)

  刀具直径 D=6mm

  第一刀:     ┌────┐
              │ ██ │  ← 刀具位置
              └────┘
  ╔═══════════════════╗
  ║███████████████████║  ← 第一刀切掉的区域
  ╚═══════════════════╝

  第二刀:          ┌────┐
                   │ ██ │  ← 偏移了 ae=3mm
                   └────┘
  ╔═════════════════════════╗
  ║█████████████████████████║  ← 两刀叠加
  ╚═════════════════════════╝
        ←ae=3mm→
```

**ae 和 Stepover 百分比的关系：**

```
ae = Stepover% × D

  50% 步距, D=6mm → ae = 0.50 × 6 = 3mm
  25% 步距, D=6mm → ae = 0.25 × 6 = 1.5mm
  75% 步距, D=6mm → ae = 0.75 × 6 = 4.5mm
```

**在公式 5 中的作用：** ae 直接影响切削力。ae 翻倍 → 切削力翻倍。

**ae 和 ap 和 fz 的区别（三个互相垂直的维度）：**

```
      Z轴（轴向）
       ↓ ap
       │
    ┌──┼──┐
    │刀│  │
    │具│  │
    │██│  │
    │██│  │
    └──┼──┘
    ╔══╪══╗
    ║  ↓ae ║ →→→ fz（进给方向）
    ║█████║ 刀具往前走
    ║█████║
    ╚═════╝

    ap = Z 方向（竖着吃多深）
    ae = 垂直于进给和Z轴（横着切多宽）
    fz = 进给方向（每齿咬多厚，沿刀具走动方向）
```

## 10. Stepover — 步距

```
符号: Stepover (也称 ae 的百分比表示)
中文名: 步距、行距、横向步距
单位: 毫米 (mm) 或 刀具直径的百分比 (%D)
```

**含义：** 相邻两刀之间的间距。控制刀具路径在 XY 平面上的重叠程度。

**和 ae 的关系：**

```
Stepover_mm = ae（同一个东西，只是叫法不同）
Stepover_pct = ae / D × 100%

Stepover = 3mm = 50% of D(6mm) → ae = 3mm
```

**不同步距的效果：**

```
粗加工 (30-50% D):
  效率高，快速去除材料
  表面粗糙（有明显的刀痕台阶）
  切削力适中

  50% D 示意:
  ┌──────┐┌──────┐
  │██████││██████│
  │██████││██████│
  └──────┘└──────┘
    ← 3mm →  重叠 3mm

精加工 (5-15% D):
  效率低，但要走很多遍
  表面光滑
  切削力很小

  10% D 示意:
  ┌──────┐
  │██████│
  │██████│
  └──┬───┘
  ┌──┴───┐
  │ ████ │
  │ ████ │
  └──────┘
    ← 0.6mm →  大面积重叠

自适应/Trochoidal (25-30% D):
  用摆线运动，切削力恒定
  可以用更大的切深补偿小的步距
  适合硬材料和低刚性机器
```

## 11. Fc — 切削力

```
符号: Fc (Cutting Force)
中文名: 切削力、主切削力
单位: 牛顿 (N)
```

**含义：** 刀具切削材料时受到的阻力。是所有切削参数的最终结果，用来做安全检查。

**计算公式：**

```
Fc = Kc × ap × ae × fz

各符号含义：
  Kc = 材料的比切削力（切掉 1mm² 面积需要多少牛顿的力）
  ap = 轴向切深（竖着切多深）
  ae = 径向切宽（横着切多宽）
  fz = 每刃进给量（每个齿咬多厚）
```

**为什么 ap 直接在公式里 — ap 不是累积出来的，是同时嵌入的深度：**

```
常见误解：
  "ap=1.5mm 是转了很多圈才慢慢切出来的，
   每转只切 fz=0.03mm，为什么力公式里直接乘 1.5mm？"

  ✗ 这个理解是错的！

实际情况：
  下刀阶段：刀通过螺旋/斜插到达 Z=-1.5mm
  水平切削阶段：刀保持在 Z=-1.5mm 不动

  在水平切削阶段：
    刀已经嵌入材料 1.5mm 深
    每转一圈，刀刃扫过的材料都是 1.5mm 高的！
    不是 0.03mm 高的！

  侧视图 — 刀在 Z=-1.5mm 水平移动时：

    Z=0 ══════════════════════
        ║  ← 材料表面
        ║
    ┌───╫───┐
    │   ║   │
    │ 刀║具 │   刀已经在 Z=-1.5mm
    │   ║   │   侧面刀刃嵌入材料 1.5mm 深
    │   ║   │
    └───╫───┘
    Z=-1.5 ══════════════════

    每转一圈，每个齿扫过这个 1.5mm 高的区域
    切掉的切屑形状：

           ← ae = 3mm →
        ┌──────────────┐  ↑
        │              │  │
        │    切屑       │  ap = 1.5mm
        │              │  │
        └──────────────┘  ↓
        │← fz = 0.03mm →│

    切屑又高(1.5mm)又宽(3mm)但很薄(0.03mm)
    切屑体积 ≈ ap × ae × fz
    切削力 Fc = Kc × ap × ae × fz
```

```
时间线 — ap 在什么时候是"累积的"，什么时候是"同时的"：

  阶段 1 — 下刀（螺旋/斜插），几秒钟
    刀从 Z=0 降到 Z=-1.5mm
    这阶段 ap 在逐渐增大：0 → 0.5 → 1.0 → 1.5mm
    力也在逐渐增大

  阶段 2 — 水平切削，几分钟（主要加工时间）
    刀保持在 Z=-1.5mm
    ap 固定 = 1.5mm ← 每转都一样！
    每转一圈：刀刃扫过 1.5mm 高 × ae 宽的材料
    力基本恒定：Fc = Kc × 1.5 × ae × fz

  阶段 3 — 抬刀，回到 Z=0

  阶段 4 — 下刀到第二层 Z=-3.0mm

  阶段 5 — 水平切削
    ap 仍然 = 1.5mm（每层的深度）
    力和阶段 2 一样

  公式算的是阶段 2 的力 — 最主要的加工状态
  ap 在这个阶段是固定的，每转都面对一样多的材料
```

```
物理含义：
  Kc × ap × ae × fz = (力/面积) × 深 × 宽 × 厚 = 力

  切屑截面积 = ap × ae × fz（近似）
  Fc = Kc × 切屑截面积
```

**力从哪来 — ap、ae、fz 是三个不同方向：**

深（ap）、宽（ae）、厚（fz）不是一回事，它们是**三个互相垂直的方向**，共同决定切屑的体积：

```
切屑（被刀刃切下来的那一小块材料）是一个长方体：

         ← ae（径向切宽）→
      ┌────────────────────┐  ↑
      │                    │  │
      │      切屑          │  ap（轴向切深）
      │                    │  │
      └────────────────────┘  ↓
      │← fz（切屑厚度）→│

三个方向互相独立、互相垂直：
  ap = Z 轴方向（竖着，上下）— 竖着吃多深
  ae = 刀具径向（横着，垂直于进给和 Z）— 横着切多宽
  fz = 进给方向（刀具平移方向）— 每个齿咬多厚

切屑体积 ≈ ap × ae × fz
切削力 Fc = Kc × ap × ae × fz
         = (力/面积) × 长 × 宽 × 高

就像算一个盒子的体积：
  体积 = 长 × 宽 × 高
  长、宽、高是三个独立维度，不是同一个东西
```

```
刀刃切入材料时，需要把材料"撕开"：

  材料越硬（Kc 大）→ 需要越大的力
  竖着吃越深（ap 大）→ 需要越大的力
  横着切越宽（ae 大）→ 需要越大的力
  每齿咬越厚（fz 大）→ 需要越大的力

  任何一个增大 → 切屑变大 → 切削力增大
  但它们影响的是切屑的不同维度，可以独立调节
```

**力的安全阈值：**

```
机器刚性决定了能承受多大的切削力：

  VERY_LOW (3018):  50 N   ← 铝罐的力就能让它抖
  LOW (X-Carve):    100 N  ← 还是比较弱
  MEDIUM (Shapeoko): 200 N ← 消费级主力
  HIGH (桌面铣床):  500 N  ← 相当结实

示例验算：
  6061铝, Kc=700, ap=1.5, ae=3, fz=0.03
  Fc = 700 × 1.5 × 3 × 0.03 = 63 N

  Shapeoko (200N 限制): 63N ÷ 200N = 31.5% ✓ 安全
  3018 (50N 限制):      63N ÷ 50N = 126% ✗ 超限！

所以 3018 必须减小参数：
  ap=0.6, ae=1.8, fz=0.018
  Fc = 700 × 0.6 × 1.8 × 0.018 = 13.6N < 50N ✓
```

**超限的后果：**

```
Fc 超过机器承受能力 → 链式反应：

  Fc 过大
    → 机器框架变形/振动
      → 刀具实际轨迹偏离预期路径
        → 尺寸精度丢失
          → 严重时刀具颤振
            → 刀具断裂
              → 工件松动飞出（危险）
```

## 12. Kc — 比切削力

```
符号: Kc (Specific Cutting Force)
中文名: 比切削力、单位切削力
单位: 牛顿/平方毫米 (N/mm²) 或 兆帕 (MPa)
```

**含义：** 切掉 1 平方毫米截面积的材料需要多大的力。是材料本身的属性。

**直观理解：**

```
Kc = 700 N/mm² (6061 铝合金)

意思是：切出一个 1mm × 1mm 截面的铝切屑
        需要 700 牛顿的力

700 N ≈ 70 公斤的力（想象一个人用 70kg 的力推刀切入材料）
```

**不同材料的 Kc：**

| 材料 | Kc (N/mm²) | 相对铝的倍数 | 切削感受 |
|------|-----------|-------------|---------|
| MDF | ~30 | ×0.04 | 像切蛋糕 |
| 硬木 | ~100 | ×0.14 | 像切木头 |
| 亚克力 | ~350 | ×0.50 | 有阻力但还行 |
| 6061 铝 | ~700 | ×1.0 | 基准 |
| 黄铜 | ~800 | ×1.14 | 和铝差不多 |
| 紫铜 | ~900 | ×1.29 | 比铝难一点 |
| 软钢 1045 | ~1500 | ×2.14 | 明显费力 |
| A36 结构钢 | ~1800 | ×2.57 | 很费力 |
| 不锈钢 304 | ~2000 | ×2.86 | 非常费力 |
| 钛合金 Ti6Al4V | ~1400 | ×2.00 | 硬+导热差，实际更难 |

**Kc 不是精确常数：**

```
Kc 会受以下因素影响：

1. 切削厚度（fz）:
   fz 越大 → 切屑越厚 → Kc 略降低（但力总量增大）
   这就是为什么 fz 太小反而不好：不仅发热，Kc 还偏高

2. 刀具磨损:
   刀刃变钝 → Kc 有效增大 → 需要更大的力

3. 切削速度（Vc）:
   一般 Vc 升高 → Kc 略降低

4. 冷却液:
   有冷却润滑 → 排屑顺畅 → Kc 有效降低

   但对于消费级简化计算，Kc 取查表值的常数已足够精确
```

**在公式 5 中的作用：** Kc 是力估算的起点。Kc 越大 → 同样参数下切削力越大 → 需要更保守的参数。

## 完整公式链路：从输入到输出

```
已知:
  材料: 6061 铝合金 → 查表: Kc=700, Vc=300-500
  刀具: 6mm 2刃硬质合金平底刀 → D=6, z=2
  机器: Shapeoko 4 → RPM上限12000, 刚性MEDIUM, 力限制200N

步骤1 — 算 RPM:
  RPM = (Vc × 1000) / (π × D)
      = (200 × 1000) / (3.14159 × 6)
      = 200000 / 18.85
      = 10610

  各符号:  Vc=200(取保守值), 1000(单位换算), π=3.14, D=6
  结果:    RPM = 10610 转/分钟

步骤2 — 算 Feed:
  Feed = RPM × z × fz
       = 10610 × 2 × 0.03
       = 637

  各符号:  RPM=10610(上一步), z=2(刀具刃数), fz=0.03(查表)
  结果:    Feed = 637 mm/min

步骤3 — 查切深上限:
  ap_max = 系数 × D
         = 0.25 × 6
         = 1.5

  各符号:  系数=0.25(MEDIUM刚性+铝查表), D=6
  结果:    ap = 1.5 mm

步骤4 — 设步距:
  ae = 50% × D
     = 0.50 × 6
     = 3.0

  各符号:  50%(粗加工标准), D=6
  结果:    ae = 3.0 mm

步骤5 — 验算力:
  Fc = Kc × ap × ae × fz
     = 700 × 1.5 × 3.0 × 0.03
     = 63

  各符号:  Kc=700(材料), ap=1.5(步骤3), ae=3.0(步骤4), fz=0.03(步骤2)
  结果:    Fc = 63 N

  检查: 63N < 200N(机器限制) ✓ 安全

最终参数:
  RPM      = 10610 转/分钟
  Feed     = 637 mm/min
  切深 ap  = 1.5 mm
  步距 ae  = 3.0 mm (50% D)
  切削力   = 63 N (31.5% 利用率)
```
