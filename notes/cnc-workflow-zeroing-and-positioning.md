# CNC 加工定位与对刀详解

> 最后更新：2026-05-16

---

## 核心问题

机器只知道"自己在哪"（机床坐标），不知道"工件在哪"。必须由人告诉机器工件的位置和方向，建立一个**工件坐标系**，加工才能开始。

```
机器的坐标 vs 工件的坐标

  机床坐标系（Machine Coordinates）：
    上电 → 回 Home → 限位开关确定机床零点
    这是机器内部的坐标，固定不变
    用户不需要关心

  工件坐标系（Work Coordinates）：
    用户定义的原点，相对于工件本身
    G-code 里写的所有坐标都是工件坐标
    机器自动换算：工件坐标 + 偏移量 = 机床坐标

    G-code:  G0 X50 Y30
    工件原点偏置: X_offset=120, Y_offset=85
    实际机床运动到: X=170, Y=115

  用户只需要设好工件原点，之后不用管机床坐标
```

---

## 1. 工件原点怎么定义

### 原点是人为选的

```
工件原点 = 图纸上 X=0, Y=0, Z=0 的位置

这个位置没有固定标准，由设计者决定。
原则：方便对刀 + 方便设计 + 方便编程。

常见的选法：

  ┌─────────────────────┐
  │ A: 左下角（最常用）  │  ← 消费级默认
  │   ×                 │
  │                     │
  │                     │
  └─────────────────────┘

  ┌─────────────────────┐
  │                     │
  │                     │
  │       ×             │  ← B: 中心点
  │                     │     适合对称零件
  └─────────────────────┘

  ┌─────────────────────┐
  │                 ×   │  ← C: 右上角
  │                     │     较少见
  │                     │
  └─────────────────────┘

  左下角最常用的原因：
    1. 刀具从左往右、从前往后移动 → 所有坐标都是正数
    2. 放置工件时，左下角最容易对齐工作台边缘
    3. CAM 软件默认原点就是左下角
```

### Z 轴原点的定义

```
Z=0 放在哪里？

  几乎所有场景都选"工件顶面"作为 Z=0：

    Z 轴
     ↑
     │  Z > 0  → 刀在工件上方（安全区域）
     │
  Z=0 ╪═════════════════  ← 工件顶面
     │
     │  Z < 0  → 刀在工件内部（切削区域）
     │
     │

  G-code 里的 Z 坐标含义：
    G0 Z5       → 抬刀到工件表面以上 5mm（安全高度，空跑）
    G1 Z-1.5    → 下刀到表面以下 1.5mm（切削）
    G1 Z-3.0    → 切到表面以下 3mm（更深）
```

### 原点是 CAD → CAM → 机器 一脉相承的

```
原点在三个环节中必须一致：

  CAD 阶段：
    设计者画图时，图纸的 (0,0) 在左下角
    ┌────────┐
    │×       │ ← 图纸原点 = 左下角
    │   设计  │
    └────────┘

  CAM 阶段：
    CAM 软件导入图纸，默认原点也在左下角
    生成的 G-code 坐标都相对于这个左下角
    G0 X10 Y10 → 从左下角往右 10mm、往上 10mm

  机器阶段：
    用户把刀移到物理工件的左下角
    按下"设为零点"
    → 机器知道：G-code 里的 X0 Y0 对应这个物理位置

  三者一致 → G-code 里写的每个坐标，机器都能正确执行
  任何一环不一致 → 加工位置偏移 → 做出来的东西歪了/偏了
```

### 常见错误

```
错误 1 — CAD 原点和 CAM 原点不一致
  CAD 里原点在中心，CAM 里默认原点在左下角
  → 加工出来的零件整体偏移

  解决：CAM 软件里可以设置原点位置，要和 CAD 一致

错误 2 — 对刀位置和 CAM 原点不一致
  CAM 里原点在左下角，但用户把刀对到了左上角
  → Y 方向镜像了

  解决：对刀前确认 CAM 软件里原点在哪里

错误 3 — 工件放歪了
  工件和机器轴线不平行
  → 加工出来的形状是歪的

  解决：用直尺/角尺对齐工件和工作台边缘
```

## 2. 机器归零（Home）

```
机器上电后，不知道自己在哪。

所有轴 → 往一个方向移动 → 碰到限位开关 → 停下
这个位置就是"机床零点"（Machine Zero）

限位开关位置是固定的，出厂时就设定好了

  Y轴限位开关
  ┌───────────────────┐
  │○                   │ ← X轴限位开关
  │    工作台          │
  │                    │
  │                   ○│ ← X轴限位开关
  └───────────────────┘
  ○ ← Y轴限位开关

Home 后，机器内部记录机床坐标 X=0, Y=0, Z=0
但这个零点和工件没有关系
```

## 3. 设工件原点（对刀 / Zeroing）

### X/Y 零点 — 确定工件在水平面的位置

```
方法 1 — 手动对刀（消费级最常用）

  1. 在控制软件里手动移动刀具（点按钮，慢速移动）
  2. 把刀移到工件左下角的上方
  3. 慢慢降 Z，让刀尖接近工件表面
  4. 用一张纸当塞尺：
     刀和工件之间夹纸 → 拖动纸张 → 感觉到轻微摩擦 → 停
  5. 按下"设为零点"按钮

              刀
              ↓
  纸 ════════╤═══════════
            ╱
  ┌────────╱───────────┐
  │ ← ×                │  ← 刀正对工件左下角
  │  工件              │     这里就是 X=0, Y=0
  └────────────────────┘

  控制软件发送：G92 X0 Y0（当前位置设为工件原点）
  或者：G10 L20 P1 X0 Y0（写入 G54 工作坐标偏置）

  精度：±0.1mm（取决于手感）


方法 2 — 边缘探测器（Edge Finder，工业用）

  一个精密工具，装在主轴上，碰到工件边缘时会跳开
  精度可以到 0.01mm
  消费级很少用，因为机器本身精度就 ±0.1mm


方法 3 — 摄像头/激光定位

  主轴旁边装个小摄像头
  在软件里看到十字线对准工件角落 → 点一下设零
  Makera Carvera 有类似功能
```

### Z 零点 — 确定工件表面的高度

```
方法 1 — 纸片法（手动，最基础）

  1. 移动刀具到工件表面上方
  2. 慢慢下降 Z 轴
  3. 一张纸夹在刀尖和工件之间
  4. 降 Z 直到纸被轻轻夹住（能抽出来但有阻力）
  5. 设 Z=0

     刀尖
      ↓
  ════╤════ 纸（~0.1mm 厚）
  ────┼──── 工件表面

  精度 ±0.1mm（取决于纸的厚度和手感）


方法 2 — Z-Probe 触碰探头（推荐，最准）

  一块已知厚度的金属片（比如 1.0mm）

  1. 把探头片放在工件表面
  2. 刀尖接触探头片 → 电路导通 → 自动检测到
  3. 机器知道：刀尖在工件表面以上 1.0mm
  4. 自动计算 Z 零点

     刀尖
      ↓
  ────┤──── 探头片（已知厚度 1.0mm）
  ────────── 工件表面

  GRBL 命令：
    G38.2 Z-10 F100    （往下探，直到接触）
    G92 Z1.0           （当前位置是 Z=1.0，即表面以上 1mm）

  精度 ±0.02mm，比手动准很多


方法 3 — 自动对刀块（高端消费级）

  固定在工作台上的对刀块
  换刀后自动下降触碰 → 自动计算刀长差
  Shapeoko 5 Pro 支持此功能
```

## 4. 工件对齐（防歪）

```
放上去的材料可能不是正的：

  ┌───────────────────┐
  │        ╱          │
  │      ╱  工件歪了   │
  │    ╱              │
  │  ╱                │
  │╱                  │
  └───────────────────┘

解决方法 1 — 物理对齐（消费级常用）
  用直尺/角尺把工件靠齐工作台的边
  假设工作台是方的，工件靠上去就是正的

解决方法 2 — 旋转坐标补偿（高级）
  1. 探测工件的两个角点
  2. 计算出歪了多少度
  3. 在 G-code 里加旋转补偿（G68）
  消费级固件（GRBL）通常不支持 G68 → 只能物理对齐
```

## 5. 完整操作流程

```
1. 开机
   打开机器电源 → 打开控制软件（UGS/Candle 等）

2. 回 Home
   机器自动移动到各轴限位开关位置 → 建立机床坐标

3. 装夹工件
   把材料固定在工作台上
   双面胶 / 压板 / 夹具
   确保工件不歪（用角尺对齐工作台边缘）

4. 装刀具
   把正确的刀具装入主轴夹头（ER11 等）
   拧紧螺母

5. 设 X/Y 零点
   手动移动刀具到工件原点位置（左下角）
   用纸片法确认位置
   控制软件点"设 X/Y 零点"

6. 设 Z 零点
   Z-Probe 触碰探头 或 纸片法
   控制软件点"设 Z 零点"

7. 空跑检查（Air Cut）
   在控制软件里把 G-code 所有 Z 值整体抬高（比如 +10mm）
   运行一遍，看刀具运动轨迹是否合理
   会不会撞到夹具？运动范围是否正常？

8. 正式加工
   恢复原始 Z 值
   开始切削
   首次加工建议站在旁边观察，随时按急停
```

## 6. G-code 中的坐标指令

```
G90 — 绝对坐标（相对于工件原点）
  G90 G0 X50 Y30 → 移动到工件坐标 (50, 30)

G91 — 相对坐标（相对于当前位置）
  G91 G0 X10 Y5  → 从当前位置往右 10mm、往上 5mm

G92 — 设当前位置为工件原点
  G92 X0 Y0 Z0  → 当前位置就是 (0,0,0)

G54-G59 — 工作坐标偏置（存储工件原点位置）
  G54 → 使用第 1 组工件坐标
  可以存储多个工件原点，方便批量加工

实际操作中：
  对刀时用 G92 或控制软件的"设零"按钮
  加工时 G-code 默认 G90（绝对坐标）
```
